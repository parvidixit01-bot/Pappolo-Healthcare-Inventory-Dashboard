# Pappolo Healthcare Inventory Management Dashboard

## Project Overview
This project analyzes healthcare inventory data to track stock levels, monitor product availability, and support better operational decisions.

## Tools Used
Power BI
Excel

## Dataset
Healthcare inventory dataset containing product categories, stock levels, suppliers, and usage patterns.

## Key Features
• Inventory stock monitoring  
• Product category analysis  
• Supplier performance tracking  
• Dashboard for real-time inventory insights

## Files in Repository
pappolo_inventory_dataset.xlsx – Raw dataset  
pappolo_inventory_dashboard.pbix – Power BI dashboard  
dashboard_screenshot.png – Dashboard preview
